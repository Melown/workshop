.. _mapproxy:

************
VTS-Mapproxy
************

`VTS-Mapproxy <https://github.com/melown/vts-mapproxy>`_ is a HTTP server that
converts non-VTS resources (raster or vector) to VTS resources (surface,
boundlayer, freelayer) on the fly.
