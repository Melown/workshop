***************
VTS-Browser-CPP
***************

`VTS JavaScript <https://github.com/melown/vts-browser-cpp>`_ is a C++ client
library which will obtain configuration from the server and display 3D data to
user on the desktop.

The `User documentation <https://github.com/melown/vts-browser-cpp/wiki>`_ is
hosted on GitHub.
