.. _vts-browser-js: 

**************
VTS JavaScript 
**************

`VTS JavaScript <https://github.com/melown/vts-browser-js>`_ is a JavaScript client
which will obtain configuration from the server and display 3D data to user.

After installation step, you may see some examples. They are heavily inspired by
the demos, `distributed along with the source code <https://github.com/Melown/vts-browser-js/tree/master/demos>`_.

The examples are also `hosted on JSFiddle <https://github.com/Melown/vts-browser-js/wiki/Examples>`_

The `API Reference documentation <https://github.com/melown/vts-browser-js/wiki>`_ is hosted on GitHub too.
